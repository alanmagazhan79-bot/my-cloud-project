☁️ Cloud Engineer Project

Облачные технологии в современной IT-индустрии

📌 О проекте

Данный проект посвящён изучению облачных технологий и профессии Cloud Engineer. В рамках работы были рассмотрены основные принципы работы облачных систем, их преимущества и перспективы развития.

⚙️ Практическая часть

В ходе проекта был создан простой веб-сайт и размещён в облаке с помощью GitHub Pages. Это демонстрирует принцип работы облачного хостинга.

📊 Исследование

Был проведён опрос, который показал, что большинство людей используют облачные сервисы, но не всегда понимают, как они работают.

🚀 Перспективы

Облачные технологии продолжают развиваться, включая направления: искусственный интеллект, большие данные и автоматизацию.
